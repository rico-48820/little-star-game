# 小星星空间探险

和独角兽一起玩转形状的儿童益智小游戏，适合 3～6 岁宝宝。

## 在线玩

开启 GitHub Pages 后，可访问：

**https://\<你的用户名\>.github.io/\<本仓库名\>/**

例如：`https://username.github.io/xiaoxingxing-game/`

## 玩法简介

按年龄选择一种模式开始游戏：

| 年龄 | 模式     | 内容说明           |
|------|----------|--------------------|
| 3岁  | 找形状   | 根据形状名称选出相同形状 |
| 4岁  | 转一转   | 根据旋转角度选出正确选项 |
| 5岁  | 在哪里   | 判断物体在场景中的位置（上/下/左/右/中间） |
| 6岁  | 拼图形   | 用给定形状联想并选出拼成的结果 |

每轮答对会得分，全部完成后根据得分获得星级；可随时点「提示」获取帮助。

## 本地运行

用浏览器直接打开仓库里的 `index.html` 即可，无需安装或构建。

## 技术说明

单文件 HTML，内联 CSS 与 JavaScript，使用 Google Fonts（Nunito）。适合部署在 GitHub Pages 等静态托管。
